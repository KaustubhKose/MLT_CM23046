Data Pipeline

Augment and batch images with ImageDataGenerator. 80/20 train-validation split.

from tensorflow.keras.preprocessing.image import ImageDataGenerator

gen = ImageDataGenerator(
    rescale=1./255,
    rotation_range=25,
    width_shift_range=0.15,
    height_shift_range=0.15,
    shear_range=0.1,
    zoom_range=0.2,
    horizontal_flip=True,
    fill_mode='nearest',
    validation_split=0.2
)

train_ds = gen.flow_from_directory(
    'fruits/', target_size=(224, 224),
    batch_size=32, subset='training',
    class_mode='categorical'
)
val_ds = gen.flow_from_directory(
    'fruits/', target_size=(224, 224),
    batch_size=32, subset='validation',
    class_mode='categorical'
)
🏗️

Build Model

Load MobileNetV2 without its top, freeze base, attach custom 3-class head.

import tensorflow as tf
from tensorflow.keras.applications import MobileNetV2
from tensorflow.keras import layers, models

# Load pretrained base (frozen)
base = MobileNetV2(input_shape=(224, 224, 3),
                   include_top=False,
                   weights='imagenet')
base.trainable = False

# Custom 3-class classifier head
inputs = tf.keras.Input(shape=(224, 224, 3))
x = base(inputs, training=False)
x = layers.GlobalAveragePooling2D()(x)
x = layers.BatchNormalization()(x)
x = layers.Dense(256, activation='relu')(x)
x = layers.Dropout(0.4)(x)
x = layers.Dense(64, activation='relu')(x)
x = layers.Dropout(0.2)(x)
outputs = layers.Dense(3, activation='softmax')(x)

model = models.Model(inputs, outputs)
print(f"Trainable params: {model.count_params():,}")
🚀

Train & Fine-tune

Phase 1 trains the head only. Phase 2 unfreezes top-30 base layers with a lower LR.

# Phase 1: Train head only
model.compile(
    optimizer=tf.keras.optimizers.Adam(1e-3),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)
cb = [
    tf.keras.callbacks.EarlyStopping(patience=4, restore_best_weights=True),
    tf.keras.callbacks.ModelCheckpoint('best_model.keras', save_best_only=True)
]
h1 = model.fit(train_ds, epochs=15, validation_data=val_ds, callbacks=cb)

# Phase 2: Fine-tune top 30 layers
base.trainable = True
for layer in base.layers[:-30]:
    layer.trainable = False

model.compile(
    optimizer=tf.keras.optimizers.Adam(1e-5),
    loss='categorical_crossentropy',
    metrics=['accuracy']
)
h2 = model.fit(train_ds, epochs=10, validation_data=val_ds, callbacks=cb)
🎯

Predict & Save

Run inference on a single image and export the final model.

import numpy as np
from tensorflow.keras.preprocessing import image

CLASS_NAMES = ['apple', 'banana', 'mango']

def classify(img_path):
    img = image.load_img(img_path, target_size=(224, 224))
    arr = image.img_to_array(img) / 255.0
    arr = np.expand_dims(arr, 0)
    preds = model.predict(arr, verbose=0)[0]
    top   = np.argmax(preds)
    return {
        'label': CLASS_NAMES[top],
        'confidence': float(preds[top]),
        'all_scores': dict(zip(CLASS_NAMES, preds.tolist()))
    }

result = classify('test.jpg')
print(result)
# → {'label': 'mango', 'confidence': 0.973, ...}

model.save('mobilenet_3class.keras')

Theory

Key Concepts

🧠

Feature Reuse

ImageNet training teaches MobileNet to detect edges → textures → object parts. These features transfer remarkably well to fruit images without any relearning.

🔒

Layer Freezing

Freezing the base prevents catastrophic forgetting of pretrained features when your dataset is small. Only your custom head learns in Phase 1.

📐

Input Size

MobileNetV2 expects 224×224×3 RGB input. All images are resized and normalized to [0, 1] before feeding. Channel order follows TensorFlow's RGB convention.

⚖️

Class Balance

Equal class sizes prevent bias. If classes are imbalanced, use class_weight argument in model.fit() to compensate during training.

🌊

Dropout

Two dropout layers (0.4 and 0.2) in the head randomly zero activations during training, forcing the model to learn redundant representations and reducing overfitting.

📉

LR Schedule

Phase 1 uses Adam at 1e-3 for fast head convergence. Phase 2 drops to 1e-5 — too high and you overwrite valuable base-layer features learned from ImageNet.
